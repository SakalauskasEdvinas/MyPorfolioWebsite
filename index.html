Top 10 Countries D3 Bar Chart

This project visualizes the top 10 countries based on either population or land area using a responsive bar chart. The data is fetched from the REST Countries API, and the chart is created using the D3.js library. Users can select between the two available filters (population or area) using a dropdown menu, and the chart updates accordingly to display the corresponding data.
